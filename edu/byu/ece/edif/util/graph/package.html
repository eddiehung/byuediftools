Provides a variety of data structures and algorithms for creating and manipulating graphs based on a Edif cell structure

The graphs supported in this package are all based on the Ptolemy graph package. This Java graph package provides a representation of both undirected and directed graphs. In addition, it provides a variety of graph algorithms for analyzing and manipulating graphs.

This package provides the following functionality:

  • A variety of graph data structures based on the EdifCell object
  • A number of algorithms for analyzing such graphs
  • The capability of generating .dot files for visualizing these graphs

The following graphs will be supported in this package:

  • DepthFirstSearchGraph This is a DirectedGraph object representing a forest of spanning trees when performing an efficient depth-first search.
  • EdifPortConnectivityGraph Represents connectivity between top-level ports of an EdifCell. This graph is created directly from an EdifCell without any other graph structures. The Nodes of this graph have as weights Collections of EdifSingleBitPort objects. The edges represent connectivity between top-level input and output ports.
